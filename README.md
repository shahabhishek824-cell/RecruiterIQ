# RecruiterIQ

RecruitIQ is a resume screening and candidate ranking web application developed using Java and Spring Boot. The system enables recruiters to upload multiple PDF resumes, extract candidate information, identify relevant skills, and rank candidates based on job requirements.

## Features

* Upload and analyze multiple PDF resumes
* Automatic skill extraction and matching
* Candidate ranking based on skill relevance
* Categorized skill filtering interface
* Resume parsing using Apache PDFBox
* Interactive web interface built with Thymeleaf

## Tech Stack

* Java 8
* Spring Boot 2.7
* Apache PDFBox
* Thymeleaf
* HTML
* CSS
* JavaScript
* Maven

## Purpose

The project was developed to simplify the recruitment process by automating resume screening and helping recruiters identify suitable candidates efficiently.
